### 0x02 名词性从句

名词可以做主语，表语，宾语，宾补，因此常见名词性从句可以分为

- 主语从句
- 宾语从句
- 表语从句



名词性从句分类

- 无意义的that + 简单句, 陈述句做名词性成分
  - that可以省略的两个原则: 不影响句子意思和不影响主从关系.
    - That he didn't show up on time is strange. 这里That不能省略，这里它引导的是从句he didn't show up on time做主语. 
- 表示“是否”的whether/if + 简单句构成名词性从句, 一般疑问句做名词性成分(变为陈述句) 
  - “是否” 只用whether
  - "如果"只用if
- 疑问词引导, 特殊疑问句作名词性成分(变为陈述句)



形式主语: 避免头重脚轻，把真正主语后置

- Its + adj/noun + that
- it + v + that



### 0x03 形容词性定语从句

分析技巧:

You are the only one whose advice he might listen to.

这里可以把这个句子分成两个

1. You are the only one
2. Your advice he might listen to

这里的后行词是Your，所以用whose来代替.  可以看到whose是做了成分的. 



| 后行词     |                                                              |      |
| ---------- | ------------------------------------------------------------ | ---- |
| Who/whom   | 如果后行词是人，就用who/that; 如果后行词是宾格，就用whom，也可以直接用who，如果后行词和介词紧密连接的情况下需要用whom. |      |
| Which/that | 后行词是表事物的名词                                         |      |
| Whose      | 后行词是his, her, its, their, your one's                     |      |
| Where      | 后行词是地点状语，通常是介词+表示地点的名词                  |      |
| When       | 后行词是时间状语，通常是介词+表示时间的名词                  |      |
| Why        | 后行词是原因状语                                             |      |



特殊用法

- Where可以代替抽象的地点状语
  - English is a "fixed-word-order"  language where each phrase has a fixed position
    - English is a "fixed-word-order"  language
    - In the language each phrase has a fixed position
- When可以代替抽象的时间状语
  - There are occasions when one must yield
    - There are occasions
    - on those occasions one must yield

- that具有名词性，它也可以做行词，此时为了避免重复，引导词需要使用which.   that which可以直接换成what. 
  - We  have that which we need



which/who还是that?

- 宜用that的情况
  - 先行词被形容词的最高形式所修饰时
  - 先行词被序数或者the very, the only, the last修饰
  - 先行词是或者被它们修饰时: much, little, none, all, few, every(thing), any(thing), no(thing)
  - 先行词既有人又有物时
  - 避免which/who重复时
- 宜用which
  - 直接放在介词后面做宾语时
  - 非限制性定语从句只能使用which/who





### 0x04 定义从句

定义从句划分为两大类:

- 限定性定语从句
  - 先行词是不确定的.
- 非限定性定语从句(描述性定语从句)
  - 先行词是唯一确定的.



定语从句的翻译:

- 限定性定语从句，通常翻译成“...的”结构，需要把描述前置
  - Life isn't some musical cartoon where you sing a little song, and your insipid dreams magically come true!  生活可不是只要唱支小曲，你那无聊的梦想就会实现的音乐卡通剧. 
- 非限定性定语从句，从句通常不需要前置，顺着翻译. 
  - He is Doctor Wei, who I have mentioned to you before.  这就是魏博士，我之前跟你提到过的. 



Remark:

- 定语从句一般紧挨着先行词，但是有时为了修辞效果，先行词和定语从句会被隔开.
  - I was the only person in my office who was invited. 
- which做关系限定词时，其后必须跟一个名词
  - It started to rain, at which point we ran. 



### 0x05 同位语从句

同位语: 一个名词或者代词对另外一个名词或者代词进行解释或者补充说明，这个名词或者代词就是同位语.



同位语从句和限定定语从句的区别: 

- 同位语从句实质上就是==名词性的从句==做定语.

  - The news that we heard yesterday spread all over the school campus.
    - 它仅仅对news描述了一下，但是不知道news是什么. 相对于从句做形容词. 
  - The news that Mr.Li will be our new English teacher is true.
    - 它突出表达了news具体到底是什么?
